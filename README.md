# ☕ Coffee Machine 

## Overview
This project is a **Python-based coffee machine simulator** that allows users to order drinks like espresso, latte, and cappuccino while managing resources and handling payments.

---

## Features
- Select drinks: espresso, latte, cappuccino
- Checks if enough ingredients are available
- Processes coins (quarters, dimes, nickels, pennies)
- Calculates and returns change
- Updates machine resources after each order
- Displays a report of resources and money
- Option to turn off the machine

---

## Technologies Used
- Python
- Object-Oriented Programming (OOP)
- Core programming concepts:
  - Functions
  - Dictionaries
  - Loops
  - Conditional statements

---

## OOP Design
- Uses classes and objects to represent components of the coffee machine
- Encapsulates logic such as:
   - Resource management
   - Payment processing
   - Drink preparation
- Improves readability, reusability, and maintainability

## How It Works
1. The user selects a drink (espresso/latte/cappuccino).
2. The machine checks if enough ingredients are available.
3. If sufficient, the user is prompted to insert coins.
4. The program checks if the payment is enough.
5. If successful:
   - Change is returned (if any)
   - Resources are updated
   - Coffee is served ☕
6. The process repeats until the user enters `"off"`.
